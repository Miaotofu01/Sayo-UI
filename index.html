浅色 / 深色主题

Sayo UI 默认深色。给 <html> 添加 data-syo-theme="light" 即可切换 GitHub Primer Light 配色。

<!-- Light theme — add to <html> --> <html data-syo-theme="light"> // Toggle at runtime document.documentElement.setAttribute('data-syo-theme', 'light'); document.documentElement.removeAttribute('data-syo-theme'); // back to dark

导航栏的主题切换按钮可以实时切换。所有 --syo-* 变量即时更新——背景、文字、边框、功能色、语法高亮全部适配。

动效 & Sayo 交互

本页所有交互都在生效——现在就可以试试。

自定义光标

全局圆环光标替代系统指针。按下放大 (CursorExpand)。光晕全程跟随。

平滑拖尾

25 点折线彗尾跟随光标。从亮/粗渐变到透明。

点击闪烁

点击位置径向脉冲——每次交互即时反馈。

弹性物理

所有卡片、标签、按钮、链接在悬停/按下时使用弹簧缓动。就是手感好。

疑难解答：光晕/拖尾不显示？

如果你看不到光晕(glow)和拖尾(trail)，最可能的原因是系统开启了"减少动效"。 Sayo UI 遵循 prefers-reduced-motion: reduce，检测到该设置时会静默跳过装饰性动效（只保留基础光标圆环以确保可用性）。

// 浏览器控制台快速诊断 matchMedia('(prefers-reduced-motion: reduce)').matches // true = 动效被跳过 !!document.getElementById('syo-cursor') // true = 光标正常 !!document.getElementById('syo-cursor-canvas') // true = 光晕正常 // Chrome DevTools → Rendering 标签 → 取消勾选 "Emulate CSS prefers-reduced-motion"

详细排查步骤见 README.md → Troubleshooting。

// Auto-inits on DOMContentLoaded // Add .syo-reveal class to elements you want animated

Scroll Spy

滚动时自动高亮侧边栏当前链接。一个属性即可启用。

本页左侧边栏就是 Scroll Spy 的实时演示——滚动页面，当前区域对应的链接会自动高亮为紫色。

Attribute 用法

<!-- 属性值 = 滚动偏移量 (px)，通常 = nav 高度 --> <aside class="doc-sidebar" data-syo-scrollspy="100"> <a href="#install">Install</a> <a href="#tokens">Tokens</a> ... </aside>

JS API

// 手动初始化 Sayo.scrollSpy.init({ nav: '.doc-sidebar', // 包含链接的容器 offset: 100, // 滚动偏移，扣除固定导航栏高度 activeClass: 'active', // 激活链接的 CSS class（默认 'active'） }); // 销毁 Sayo.scrollSpy.destroy();

工作原理

模块查找容器内所有 <a href="#..."> 链接，定位对应目标元素，在 scroll 事件中判断当前处于哪个区域，为对应链接添加 active class。配合 CSS 的 .active 样式实现紫色左边框 + 紫底高亮效果。

属性自动初始化

在 <body> 或任意容器上加 data-syo-* 属性，无需写 JS。

Attribute Effect
data-syo-cursor Enable global custom cursor + glow
data-syo-trail Enable global smooth trail
data-syo-ripple Mark container for click ripple (global flash always on)
data-syo-parallax Element shifts with mouse in parent (value = max px)
.syo-reveal Scroll-triggered staggered entrance (CSS class)
<!-- The page you're looking at right now --> <body data-syo-cursor data-syo-trail> <section data-syo-ripple> <h1 class="syo-reveal">Hello</h1> </section> </body>

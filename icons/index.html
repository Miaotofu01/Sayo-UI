Logo — 六版方案对比

v1 同心圆·精炼 逼近圈 → 命中圈 → 精准命中。经典三环，节奏收紧。 logo-v1-rings-refined.svg
v2 S 字母几何 "S" for Sayo。四段弧线拼接，平滑曲线感。右上角点缀。 logo-v2-s-monogram.svg
v3 游标十字环 Sayo 自定义光标浓缩为符号。外环 + 十字准星 + 中心点。 logo-v3-cursor-crosshair.svg
v4 音律节拍 音频可视化 / 节拍柱状图。从左到右渐强，节奏感。 logo-v4-rhythm-bars.svg
v5 重叠透镜 ← 当前 两圆交叠形成透镜。Primer × Sayo 的交集——形式与动效的融合。 logo-v5-overlap-lens.svg
v6 单环斜切 极简。一环 + 一刀。负空间隐约构成 "S"。果断，不废话。 logo-v6-ring-slash.svg
